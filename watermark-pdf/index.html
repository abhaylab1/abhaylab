How it works

Your PDF is parsed and modified entirely in the browser using an open-source PDF library — the watermark text is drawn directly onto each page at the size, rotation, opacity and color you choose, then the file is rebuilt and handed straight to your downloads folder.

Encrypted or password-protected PDFs may fail to open; remove the password first if you hit an error.
